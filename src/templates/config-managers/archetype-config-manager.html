变体生成器配置

配置变体生成系统使用的AI模型。变体生成分为两个阶段：设计蓝图和一次性生成所有专长。

AI 模型配置

模型推荐：设计模型推荐使用创造力强的模型（如 grok-beta、claude-sonnet-4），生成模型推荐使用 Function Calling 支持好的模型（如 gpt-4o）

用于设计变体蓝图（画风、机制、专长方向）的AI模型（推荐: grok-beta 创造力强，gpt-4o 平衡性好）
用于一次性生成所有专长完整数据的AI模型（推荐: gpt-4o Function Calling支持好）

生成流程：

  1. 阶段1 - 设计蓝图：AI设计变体整体方案（名称、画风、机制、所有专长的设计方向）
  2. 阶段2 - 一次性生成：AI在同一上下文中生成所有专长的完整JSON数据，确保画风和机制一体性
  3. 阶段3 - 格式化存储：自动校验、修复数据，存入变体文件夹
