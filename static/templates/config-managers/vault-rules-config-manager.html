更好的属性 暂时停用

此功能目前暂时停用以修复已知问题，下方设置已禁用且不会生效。

启用后，角色的敏捷、智力和魅力属性将获得额外效果（仅限玩家角色）
武器打击重击时，结果视为获得敏捷调整值一半（向下取整）的无名加值，降低重击门槛
法术打击重击时，结果视为获得智力调整值一半（向下取整）的无名加值，降低重击门槛
投掷豁免的大失败判定时，结果视为获得魅力调整值一半（向下取整）的无名加值，降低大失败几率
