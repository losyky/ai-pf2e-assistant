图标生成配置

配置AI自动生成图标的功能。需要在API配置中设置图像生成API才能使用此功能。

生成开关

启用后将为生成的物品和怪物自动生成图标

生成参数

输入用于生成图标的AI模型名称。不同模型的风格、速度和成本各不相同
生成图标的尺寸（如：1024x1024, 512x512等）。注意：不同模型支持的尺寸选项可能不同
生成图标的艺术风格（如：fantasy art, pixel art, realistic等）

使用说明

  • 图标生成需要配置图像API（在"API配置管理"中设置）
  • 不同模型支持不同的尺寸选项，请根据实际API选择
  • 图标生成会消耗额外的API调用次数
  • 生成的图标会自动应用到新创建的物品上
